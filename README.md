# Hyper-V toolset on Alpine Linux VM:

 1. Get script:
    ```
    wget https://raw.githubusercontent.com/Luunoronti/Tools/master/alpine-vm-setup-hyperv.sh
    ```
3. Make it executable:
    ```
    chmod +x setup-hyperv.sh
    ./setup-hyperv.sh
    ```
4. Execute
    ```
    ./setup-hyperv.sh
    ```
